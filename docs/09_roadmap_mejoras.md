# 09. Roadmap De Mejoras

## Alta prioridad
1. Seguridad Supabase (RLS por tablas/roles)
2. Autorizacion robusta por endpoint
3. Tests backend/frontend para flujos criticos
4. Migraciones versionadas de base de datos
5. Manejo de errores y logging estructurado

## Media prioridad
1. CI/CD en GitHub Actions
2. Documentacion tecnica ampliada en README raiz
3. Optimizacion de rendimiento en consultas y dashboard

## UX/producto
1. Dashboard con KPI accionables
2. Alertas de inventario bajo/caducidad
3. Mejoras de experiencia en flujo de venta
