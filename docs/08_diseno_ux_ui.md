# 08. Diseno UX/UI

## Estado actual
La app tiene las pantallas de negocio base y estilos por pagina.

## Objetivo UX
Pasar de interfaz funcional a interfaz orientada a operacion diaria:
- Menos friccion para registrar ventas
- Alertas claras de stock bajo/caducidad
- Dashboard accionable

## Directrices de diseno recomendadas
- Definir design system (tokens: color, tipografia, espaciado)
- Unificar componentes (tablas, formularios, modales, estados de carga/error)
- Jerarquia visual fuerte para KPI y alertas
- Mejorar accesibilidad (contraste, focus, teclado)
- Responsive real para tablet/mostrador

## Prioridad de rediseño
1. Dashboard
2. Productos
3. Ventas
4. Compras/Proveedores
