# 01. Contexto Y Objetivos

## Proyecto
Sistema de gestion de inventario y ventas con:
- Frontend Angular (SPA)
- Backend FastAPI
- Base de datos en Supabase (PostgreSQL)

## Objetivo funcional
Permitir operaciones de negocio de un comercio:
- Gestion de productos y stock
- Registro de ventas y detalle de ventas
- Gestion de proveedores
- Gestion de empleados (roles Admin/Empleado)
- Gestion de compras/reposicion
- Dashboard con indicadores
- Control de caducidad

## Alcance actual
El proyecto ya dispone de estructura completa frontend/backend y rutas para los modulos principales.
